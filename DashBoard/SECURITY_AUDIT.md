# تقرير الأمان — PlayJaramanaa

تاريخ المراجعة: **2026-05-20**

## الملخص

نموذج الترخيص **offline + Ed25519** مُنفَّذ بشكل صحيح: التوقيع على الـ payload، التحقق بالمفتاح العام المُضمَّن في التطبيق، ربط التوكن بـ `installationId` و`deviceHintsHash` و`buildConstraint`.

الحماية هنا **ردع وتجاري** وليست DRM: على جهاز **root** أو APK مُعدَّل يمكن تجاوز القيود. الهدف منع الاستغلال العادي وتسريب المفتاح الخاص.

---

## ما تم إصلاحه في هذه الجولة

| المنطقة | الإجراء |
|---------|---------|
| التحقق من الترخيص | رفض `signatureVersion` غير المدعوم |
| PIN المالك | PBKDF2-HMAC-SHA256 (120k iterations) + دعم الهاش القديم |
| التخزين الآمن | `encryptedSharedPreferences` على Android |
| النسخ الاحتياطي | `android:allowBackup="false"` |
| أخطاء الإقلاع | إخفاء stack trace في `release` |
| لوحة التحكم | المفتاح الخاص في `sessionStorage` فقط (ليس `localStorage`) |
| سجل التوكنات | لا يُخزَّن التوكن الكامل ولا المفتاح الخاص |
| CSP | سياسة محتوى في `index.html` |
| توثيق | هذا الملف + `LICENSE_KEYS.md` |

## مخاطر متبقية (مقبولة أو تحتاج عملية)

### عالية — عملية تشغيل

1. **المفتاح الخاص في المتصفح**  
   لوحة التحكم ما زالت تدعم التوقيع من الواجهة لراحة التطوير. للإنتاج: استخدم `playtime/tool/dev_license_tool.dart` أو `DashBoard/main.py` على جهاز معزول فقط.

2. **توقيع Release بمفتاح debug**  
   APK الحالي موقّع بمفتاح debug (مناسب للتجربة). قبل المتجر: أنشئ keystore إنتاج في `android/app/build.gradle.kts`.

### متوسطة — تصميم

3. **جهاز root / APK معدّل** — لا يوجد كشف root؛ يمكن تعديل SQLite أو تخطي `LicenseVerifier`.
4. **حد المحطات** — يعتمد على ترتيب الفرز في الواجهة وليس قيداً مشفراً لكل محطة.
5. **تلاعب الساعة** — كشف الرجوع > 3 دقائق فقط؛ `tamperDetected` في SQLite قابل للتعديل على جهاز مخترق.

### منخفضة

6. المفتاح العام قابل للاستخراج من APK (متوقع).
7. لا obfuscation بعد — يُنصح بـ `flutter build apk --obfuscate`.

## قائمة تحقق للنشر

- [ ] عدم رفع `LICENSE_KEYS.md` أو `keys.env` إلى Git
- [ ] تشغيل لوحة التحكم على `localhost` فقط
- [ ] بناء APK بـ `--dart-define=LICENSE_PUBLIC_KEY_B64=...` المطابق لملف المفاتيح
- [ ] keystore إنتاج قبل Google Play
- [ ] `--obfuscate --split-debug-info=build/symbols` للإصدار النهائي
- [ ] نسخ احتياطي للمفتاح الخاص خارج الجهاز (مدير كلمات مرور / USB مشفر)
